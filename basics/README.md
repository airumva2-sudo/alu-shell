# basics

ALU "Shell, basics" project. Each script is exactly two lines:
`#!/bin/bash` followed by a single command. No backticks, &&, ||, or ;.

## Files

| File | Description |
| ---- | ----------- |
| 0-current_working_directory | Print the absolute path of the current working directory |
| 1-listit | List contents of the current directory |
| 2-bring_me_home | Change working directory to the user's home directory |
| 3-listfiles | Long-format listing of current directory |
| 4-listmorefiles | Long-format listing including hidden files |
| 5-listfilesdigitonly | Long format, numeric UIDs/GIDs, including hidden files |
| 6-firstdirectory | Create /tmp/my_first_directory |
| 7-movethatfile | Move /tmp/betty into /tmp/my_first_directory |
| 8-firstdelete | Delete /tmp/my_first_directory/betty |
| 9-firstdirdeletion | Delete /tmp/my_first_directory |
| 10-back | Change directory to the previous one |
| 11-lists | Long listing of ., .., /boot including hidden files |
| 12-file_type | Print the type of /tmp/iamafile |
| 13-symbolic_link | Create symbolic link __ls__ to /bin/ls |
| 14-copy_html | Copy newer/missing .html files to parent directory |
| 15-lets_move | Move files starting with uppercase letter to /tmp/u |
| 16-clean_emacs | Delete files ending with ~ |
| 17-tree | Create welcome/to/school in one mkdir |
