# loops_conditions_and_parsing

ALU Bash scripting project covering loops, conditions, and parsing.

## Description

This directory contains shell scripts written in Bash that demonstrate
loops (for, while, until), conditional statements (if, elif, else, case),
and text parsing using standard Unix tools (cut, awk, sort, uniq).

All scripts:
- Start with `#!/usr/bin/env bash`
- Are executable
- Pass shellcheck where applicable

## Files

| File | Description |
| ---- | ----------- |
| 1-for_best_school | Loops "Best School" 10 times with a for loop |
| 2-while_best_school | Loops "Best School" 10 times with a while loop |
| 3-until_best_school | Loops "Best School" 10 times with an until loop |
| 4-if_9_say_hi | Loops 1..10 and says "Hi" on 9 |
| 5-4_bad_luck_8_is_your_chance | Loops 1..10 with bad/good luck branches |
| 6-superstitious_numbers | Lists 1..20 with superstition annotations |
| 7-clock | Displays hours, minutes, seconds |
| 8-for_ls | Lists files in current dir, formatted |
| 9-to_file_or_not_to_file | Tells you about the file "school" |
| 10-fizzbuzz | Classic FizzBuzz 1..100 |
| 11-read_and_cut | Cuts fields 1,3,6 from /etc/passwd |
| 12-tell_the_story_of_passwd | Pretty-prints /etc/passwd |
| 13-lets_parse_apache_logs | Apache log → IP and HTTP status |
| 14-dig_the-data | Apache log → counted, sorted IP+status pairs |
